## типи даних:
### byte - це 8-розрядні цілі числа, зі значеннями в діапазоні від -128 до 127
### short - це 16-розрядні цілі числа, зі значеннями в діапазоні від -32768 до 32767
### int - це 32-розрядні цілі числа, зі значеннями в діапазоні від -2^31 до 2^31-1
### long - це 64-розрядні цілі числа, зі значеннями в діапазоні від  -2^63 до 2^63-1
### float - Одинарна точність - +- 1.4 * 10^-45 до +- 3.4 * 10^38$ - Близько 6-7 десяткових знаків
### double - Подвійна точність - +- 4.9 * 10^-324 до +- 1.8 * 10^308 - Близько 15-16 десяткових знаків
### char - для символів, представлених 16-бітовим стандартом Unicode
### boolean - true or false
